# Project Management System

A web-based Project Management System developed to manage projects, tasks, and users efficiently using Django.

## Features
- User registration and authentication
- Project and task management
- Organized workflow for tracking project progress
- Structured data storage using SQL database

## Technologies Used
- Python
- Django
- HTML, CSS
- SQLite (SQL)

## Project Structure
- core – Main configuration and settings
- register – User registration and authentication
- projects – Project and task management modules
- manager – Management and coordination logic

## Purpose
This project was developed to understand real-world project coordination, backend logic, and database-driven web applications.
